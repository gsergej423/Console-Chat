# ConsoleChat

ConsoleChat — это консольное приложение для обмена сообщениями на языке C#, разработанное в учебных целях для демонстрации реализации аутентификации пользователей, локального управления чатами и базового сетевого взаимодействия между компьютерами.

Проект поддерживает регистрацию и вход пользователей, автоматическое создание личного чата **Избранное**, создание личных и групповых чатов, базовые функции управления чатами, а также файловое хранение данных. Кроме того, приложение реализует простое TCP/IP-соединение для отправки текстовых сообщений, файлов, фотографий и видео между подключёнными компьютерами.

## Обзор проекта

Цель проекта — продемонстрировать практическое применение:

- архитектуры консольного приложения;
- логики аутентификации пользователей;
- хранения данных в файловой системе;
- управления личными и групповыми чатами;
- локального хранения истории сообщений;
- сетевого взаимодействия между компьютерами по протоколу TCP/IP.

## Основные возможности

- регистрация и вход пользователей;
- автоматическое создание личного чата **Избранное** для каждого пользователя;
- создание личных чатов между двумя пользователями;
- создание групповых чатов с несколькими участниками;
- отправка сообщений в активный чат;
- удаление и переименование чатов;
- приглашение новых участников в групповые чаты;
- TCP/IP-подключение между компьютерами;
- отправка текста, файлов, фотографий и видео.
