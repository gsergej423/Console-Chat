# ConsoleChat

**ConsoleChat** — это консольный чат на C#, в котором реализованы:

- регистрация и вход пользователей;
- чат **Избранное** по умолчанию для каждого пользователя;
- создание чатов на 2 человека;
- создание групповых чатов;
- отправка сообщений в чат;
- удаление и переименование чатов;
- приглашение новых участников в группу;
- сетевое подключение между компьютерами;
- отправка текста, файлов, фото и видео.

## Назначение проекта

Проект создан как учебное консольное приложение для демонстрации работы:

- с консольным интерфейсом;
- с файлами;
- с пользовательскими профилями;
- с логикой чатов;
- с сетевым обменом данными.

## Возможности

ConsoleChat позволяет:

- создавать пользователей;
- автоматически создавать чат **Избранное**;
- создавать личные и групповые чаты;
- отправлять сообщения в выбранный чат;
- управлять чатами;
- подключаться к другому компьютеру по сети;
- передавать разные типы данных.
